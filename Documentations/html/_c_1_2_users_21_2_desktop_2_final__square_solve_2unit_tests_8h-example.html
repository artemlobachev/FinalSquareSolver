Calculates the factorial of a number.This function computes the factorial of a given non-negative integer. It uses a recursive approach to calculate the factorial.

Parameters
nThe number for which the factorial is to be calculated.
Returns
The factorial of the number.

The factorial of a non-negative integer n is the product of all positive integers less than or equal to n. For example, the factorial of 5 is 5 * 4 * 3 * 2 * 1 = 120. This function uses recursion to compute the factorial.
